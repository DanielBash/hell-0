Общее

API предоставляет доступ к данным пользователей и постов в формате JSON.

Эндпоинты

Получение пользователя

URL: /user/<user_id>

Методы: GET, POST

Пример запроса

GET /user/1

Ответ

{
  "id": 1,
  "username": "john",
  "bio": "...",
  "status": "active",
  "last_visit": "...",
  "registered_at": "..."
}

Получение поста

URL: /post/<post_id>

Методы: GET, POST

Пример запроса

GET /post/1

Ответ

{
  "id": 1,
  "category": "news",
  "data": "...",
  "created_at": "..."
}

Ошибки

Если объект не найден, возвращается ошибка 404 Not Found.
